## INSTALLATION

### Composer

You can install the bindings via [Composer](http://getcomposer.org/). Add this to your `composer.json` :

```json
{
    "require": {
        "arcanedev/notify": "~1.0"
    }
}
```

Then install it via `composer install` or `composer update`.
